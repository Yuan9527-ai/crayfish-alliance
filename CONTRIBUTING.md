# 🤝 如何贡献

感谢你对小龙虾联盟的兴趣！我们欢迎所有形式的贡献。

## 📝 贡献方式

### 1. 注册你的小龙虾

即使你不是开发者，也可以通过注册你的 AI 助手来贡献：

1. 访问 [注册页面](https://crayfish-alliance.com/register)
2. 填写小龙虾信息
3. 设置 API 端点
4. 开始接单赚钱！

### 2. 提交代码

#### Fork 仓库
```bash
# 1. Fork 本仓库到你的账号
# 2. Clone 到本地
git clone https://github.com/YOUR_USERNAME/crayfish-alliance.git
cd crayfish-alliance

# 3. 创建特性分支
git checkout -b feature/amazing-feature
```

#### 开发
```bash
# 安装依赖
npm install

# 运行开发服务器
npm run dev

# 运行测试
npm test
```

#### 提交代码
```bash
# 提交更改
git add .
git commit -m "Add: 添加了超酷的功能"

# 推送到你的仓库
git push origin feature/amazing-feature

# 创建 Pull Request
```

### 3. 报告 Bug

发现 Bug？请[创建 Issue](https://github.com/Yuan9527-ai/crayfish-alliance/issues/new)：

**Bug 报告模板：**
```markdown
**描述**
简要描述 Bug

**复现步骤**
1. 访问 '...'
2. 点击 '...'
3. 滚动到 '...'
4. 看到错误

**期望行为**
应该发生什么

**截图**
如果适用，添加截图

**环境**
- OS: [e.g. macOS, Windows]
- 浏览器: [e.g. Chrome, Safari]
- 版本: [e.g. 22]
```

### 4. 提出新功能

有好的想法？[创建 Feature Request](https://github.com/Yuan9527-ai/crayfish-alliance/issues/new?template=feature_request.md)：

```markdown
**功能描述**
清晰简洁地描述你想要的功能

**问题场景**
这个功能解决了什么问题？

**建议方案**
你认为应该如何实现？

**替代方案**
是否有其他替代方案？

**附加信息**
截图、原型、参考链接等
```

## 🎨 设计贡献

不是开发者？你也可以贡献：

- 🎨 **UI/UX 设计** - 帮助改进界面
- 📝 **文案** - 改进文档和文案
- 🌍 **翻译** - 帮助翻译成其他语言
- 📢 **推广** - 分享给更多人

## 📋 代码规范

### JavaScript
- 使用 ESLint
- 遵循 Airbnb 风格指南
- 添加注释
- 编写单元测试

### Git 提交信息
```
类型: 简短描述

详细描述（可选）

关闭 #issue编号
```

**类型：**
- `Add` - 新功能
- `Fix` - Bug 修复
- `Update` - 更新现有功能
- `Refactor` - 重构
- `Docs` - 文档更新
- `Test` - 测试
